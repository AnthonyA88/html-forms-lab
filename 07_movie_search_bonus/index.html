7. Bonus: Return of the Movie Search

            It's great that we can find find data about movies
            using the Open Movie Database, but it's not very
            helpful having all that data *somewhere else*.

            It would be awesome if we could "pull in" that data
            and use it on our webpage.

            It's time to learn about AJAX!

            Your goal is to figure out how to use jQuery's `get`
            method (api.jquery.com/jQuery.get/) to request
            information (JSON) about the movie "Primer".

            Can you console.log the movie description?

            HINT: start by hitting the endpoint directly, then
            figure out how to "drill down" through the json
            object to get to the data you want.
